StablexUI integration with HML
==============================

Source xml can be found at `hml/xml/Alerter.xml`

Running this example:  
# `haxe generate.hxml`
# `openfl test flash`

You can see results here:  
* [Flash](http://ui.stablex.ru/v2/hml/flash/)
* [Html5](http://ui.stablex.ru/v2/hml/html5/)
